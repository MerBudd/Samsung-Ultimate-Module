The Ultimate S23 module ports features which are exclusive to newer devices, to S23 series devices; such as Now brief, wallpaper AOD, Audio Eraser, etc.

> [!NOTE]
> There is also an Ultimate module for S22 series devices. Please see the [S22 branch](/../S22X/) of this repo for the Ultimate S22 module.

## Supported devices
> [!IMPORTANT] 
> The Ultimate S23 module currently supports these devices:
> * S23 (SM-S911 / dm1q)
> * S23+ (SM-S916 / dm2q)
> * S23 Ultra (SM-S918 / dm3q)

## Features
The Ultimate module adds general features, camera features and AI features which are normally exclusive to newer devices.  
It also adds some One UI 8.5 apps, China-exclusive features, and spoofs.

Please see [FEATURE-LIST.md](/FEATURE-LIST.md) for more information and a detailed list of all features added.
